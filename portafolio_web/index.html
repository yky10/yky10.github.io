Analizador lexico, sintactico y semantico

React/Axios

Fecha: 31/05/2024

drive: https://drive.google.com/drive/folders/1btm8PpdCCmeJkNLpuktzhu07pV-vtQ7p

Ver Funcionamiento

En el curso de compiladores, el analizador léxico, sintáctico y semántico son componentes clave del proceso de compilación. El analizador léxico convierte el código fuente en una secuencia de tokens, que son las unidades básicas de significado (como palabras clave, identificadores y operadores). Luego, el analizador sintáctico toma estos tokens y construye una estructura jerárquica (normalmente un árbol de derivación o de análisis) para verificar que la secuencia de tokens siga las reglas gramaticales del lenguaje de programación. Finalmente, el analizador semántico revisa que dicha estructura sintáctica tenga sentido dentro del contexto del programa, verificando aspectos como tipos de datos, alcances de variables y operaciones válidas. Estos tres analizadores trabajan en conjunto para asegurar que el código fuente sea correcto antes de ser traducido a código máquina.
